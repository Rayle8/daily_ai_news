## 🔥 重大发布

### Anthropic 的 Claude 现在可以控制你的 Mac，推进实用AI代理建设竞赛

**摘要**: Anthropic 发布了重大更新，允许 Claude 直接控制用户的 Mac 电脑，将聊天机器人转变为可以独立工作的自主数字操作员。

**要点**:
- 更新允许 Claude 代表用户点击按钮、打开应用程序、填写表单和浏览软件
- 现已对付费订阅者开放研究预览
- 同时集成到 Claude Cowork（生产力工具）和 Claude Code（开发者命令行代理）中
- 将移动版"Dispatch"功能（从手机分配任务）首次扩展到 Claude Code
- 创建了端到端工作流：用户在移动端发出指令，回来就能拿到完成的任务

**影响**: 这是迄今为止最雄心勃勃的消费级AI代理发布之一，超越了对话交互，走向真正的自主计算机控制，可能从根本上改变用户与设备的交互方式。

📅 **来源**: VentureBeat • 2026年3月24日

**摘要**: OpenAI 宣布关闭其强大的文生视频模型 Sora，距离向公众开放有限访问仅数月时间。

**影响**: 突然关闭引发了关于 OpenAI 战略优先级的疑问，也让人怀疑初始炒作过后独立生成视频产品的商业可行性。

## 🔬 研究论文

### xMemory 将AI代理的Token成本和上下文膨胀降低了50%

**摘要**: 研究人员开发了名为 xMemory 的新型内存管理技术，可以显着减少多会话AI代理中的Token使用和上下文膨胀。

**要点**:
- 用四级语义层次结构组织对话主题，取代传统的扁平RAG
- 实验中，每个查询的Token使用从9000+减少到约4700
- 在各种LLM模型上都提高了回答质量和长程推理能力
- 由伦敦国王学院和艾伦·图灵研究所的研究人员开发

**影响**: 这种技术可以让长期存在的AI助手更经济高效地部署，在多个会话间保持连贯记忆，而不会让计算成本呈指数增长。

📅 **来源**: VentureBeat • 2026年3月25日  
🔗 **链接**: https://venturebeat.com/orchestration/how-xmemory-cuts-token-costs-and-context-bloat-in-ai-agents/
