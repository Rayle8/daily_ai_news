重大发布

Google 发布 TurboQuant 算法，KV缓存内存压缩最高达8倍，推理成本砍半

Google Research 发布了全新的 TurboQuant 算法，解决了大语言模型长期面临的 Key-Value 缓存瓶颈问题。新技术可以将KV缓存内存使用平均减少 6倍，注意力对数计算性能提升 8倍，企业部署大模型推理成本可降低 50%以上。

  • 该算法完全基于软件优化，可直接应用于现有模型和硬件
  • 发布后24小时内，社区已经将其移植到 MLX（Apple Silicon）和 llama.cpp 等流行本地AI库
  • 研究论文和算法代码已公开，可免费用于企业和学术用途

来源：VentureBeat

Anthropic 推出 Claude Mac 控制功能，AI可直接操作你的电脑

Anthropic 发布了重大更新，让 Claude 获得了直接控制 Mac 电脑的能力：AI 可以点击按钮、打开应用、输入文字、在软件中导航，真正实现了"给 Claude 派活，你去休息"的代理工作流。

  • 该功能已作为研究预览向付费订阅用户开放
  • 同时覆盖 Claude Cowork（生产力工具）和 Claude Code（开发工具）
  • 将原本仅在 Claude Cowork 中提供的 Dispatch 功能（从手机分配任务）扩展到了 Claude Code
  • 这被业内视为是构建真正可工作AI代理进程中最激进的一步

来源：VentureBeat

产业动态

音乐行业悄悄拥抱AI：制作人承认超一半嘻哈采样已由AI生成

尽管音乐行业对外对AI态度谨慎，但实际上AI已经深度渗透到音乐创作中。业内制作人 Young Guru 透露，在采样嘻哈领域，超过一半的放克灵魂采样现在已经由AI生成，而不是再去授权原始音乐或雇佣乐手。

  • 乡村音乐、流行音乐领域也普遍存在类似情况
  • 歌曲作者 Michelle Lewis 表示，没有人愿意公开承认使用AI创作
  • AI 让制作人可以快速实验编排，生成样例素材，降低创作门槛

来源：The Verge

Anthropic 遭遇安全泄露，下一代模型 "Mythos" 信息意外曝光

Anthropic 发生了一起安全事故，一个不安全的数据存储库泄露了公司内部信息，包括下一代大模型 Mythos 的存在，以及一场仅限CEO邀请参加的活动细节。Fortune 最先报道了这一事件。

目前 Anthropic 尚未官方确认 Mythos 模型的发布时间和具体规格。

研究进展

xMemory 新技术大幅削减AI代理 Token 消耗，解决上下文膨胀问题

新研究提出的 xMemory 技术通过将扁平的 RAG 替换为四层语义层次结构，将多会话AI代理的 Token 使用量减少了近一半，有效缓解了上下文膨胀问题。

这项技术可以帮助AI代理保持更长会话的一致性，同时降低运行成本。

来源：VentureBeat
