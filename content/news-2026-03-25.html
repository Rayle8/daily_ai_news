Anthropic 的 Claude 现在可以控制你的 Mac

概要：Anthropic 推出了目前最雄心勃勃的消费级AI代理功能，让 Claude 可以直接控制用户的 Mac 电脑。

  • Claude 现在能自主点击按钮、打开应用、输入文字、操作软件
  • 付费订阅用户可立即体验研究预览版

  • 将上周推出的 Dispatch 功能（从手机分配任务）扩展到 Claude Code
  • 实现了从任何地方下发指令，回来就能拿到成品的端到端流程

影响：这标志着AI代理从对话阶段进入到真正能自主完成工作的新阶段，加速了实用型AI代理的竞争。

Cloudflare 推出 Dynamic Workers：AI代理代码执行快100倍

概要：Cloudflare 推出了全新的轻量级沙箱系统 Dynamic Workers，抛弃传统容器，让AI代理代码运行速度提升100倍。

要点：

  • 毫秒级启动，仅占用几MB内存
  • 可与发起请求在同一线程运行
  • 相比传统Linux容器：启动快100倍，内存效率高10-100倍
  • 符合Cloudflare的"Code Mode"理念：让LLM直接写代码调用API，比顺序调用工具更高效
  • 将MCP服务器转成TypeScript API可减少81%的token使用
  • 定价：每独特Worker每日$0.002，外加CPU和调用费用

影响：Dynamic Workers 可以显著降低企业大规模部署AI代理的基础设施成本并提升性能。
